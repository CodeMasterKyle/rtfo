To register in the game, you need to choose a game client

The server has auto-registration enabled

Account registration takes place in the game itself. You must launch the game client, enter your desired login and password; if the login is not taken, you will be taken to the server list. Choose your server carefully; if you are playing with the Interlude client, you need servers with the Interlude prefix, if with Gracia Final, then with the Gracia Final prefix, and so on.

All files are packed in a .zip archive
