Tentang proyek

Selamat datang! Kami adalah server game yang kompleks!

Kronik dan klien

Kami sedang mengerjakan platform ini Selingan dan Final Gracia, tapi ini jauh dari klasik dalam pengertian modern.
Menunggumu pelanggan yang stabil dan konten yang kaya, kami telah meninggalkan pengaya dan membuat server unik untuk setiap selera..
Server stabil, kurangnya donasi, perhatian terhadap detail tidak akan membuat Anda bosan.

Kami sedang tampilterhadap KP dan klan yang dibeli.
Kami memiliki hanya pemain sungguhan, tidak ada donasi, dan persaingan yang sehat.

Administrasi Saya tidak pernah berdagang "dalam bayang-bayang" dan dia tidak punya rencana untuk melakukannya.
Kami terbuka untuk berdialog, bekerja secara aktif dengan komunitas, dan memperlakukan setiap pemain dengan hormat.
Server mereka secara teratur dipromosikan di sumber daya teratas

Untuk mendaftar dalam game, Anda harus memilih klien game

Pendaftaran otomatis diinstal di server

Pendaftaran akun dilakukan di dalam game itu sendiri. Anda harus meluncurkan klien game, memasukkan nama pengguna dan kata sandi yang diinginkan, jika login tidak terisi, Anda akan dibawa ke daftar server. Pilih server dengan hati-hati, jika Anda bermain dengan klien Interlude, maka Anda memerlukan server dengan awalan Interlude, jika dengan Gracia Final, maka dengan awalan Gracia Final, dan seterusnya.

Semua file dikemas dalam .arsip zip

Klien Gracia Final dan Interlude sudah dikonfigurasi untuk server kami.

Jika Anda sudah memiliki klien game Gracia Final atau Interlude, unduh patch kami.

Semoga Anda mendapatkan permainan yang menyenangkan di proyek kami
